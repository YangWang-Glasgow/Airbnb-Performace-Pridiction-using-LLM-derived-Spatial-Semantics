# Task: Fix and Format JSON

You are a JSON processing expert. Identify and correct any misconstructed JSON in the input text. Your goal is to produce a corrected JSON that can be successfully parsed using Python's `json.loads()` function.

## Instructions:

1. Identify JSON:

* Extract the JSON from the input text, ignoring surrounding text.

2. Correct Errors:

* Fix any syntax errors (e.g., missing commas, incorrect quotes, unmatched brackets)

* Ensure all keys and string values are enclosed in double quotes (")

* Do not alter the content or values in the JSON.

3. Output Requirements:

* Provide only the corrected JSON, formatted to the following structure:

```json
[
 {
  "specific_locations": [<str>, <str>, ...],
  "general_references": [<str>, <str>, ...]
}
]
```

* Ensure it is a valid JSON and can be parsed using json.loads()
